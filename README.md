# Three.js Portfolio

Inspired by Bruno Simon's **Three.js portfolio**

## Installing

1. Install dependencies

```bash
npm install
```

2. Start it

```bash
npm run dev
```

6. Visit [http://127.0.0.1:8080](http://127.0.0.1:8080)

You should see a house with full of interesting things!
